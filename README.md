A code sandbox react 16.3.2 project where I play around with some cool react features.

```
- Render Props Pattern
- Portals
- Context API
- New LifeCycle
- AsyncMode
- Higher Order Component Pattern
```

#### React Libs
```
- StyledComponents
```
